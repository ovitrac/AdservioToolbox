# Cheat Sheet — Level 1: Daily Essentials

## Claude Code — Launch & Session

| Action | Command |
|--------|---------|
| Start a session | `claude` |
| Start with prompt | `claude "explain this function"` |
| Resume last session | `claude --resume` |
| Resume by ID | `claude --resume <session-id>` |
| Continue most recent | `claude -c` |
| Non-interactive (print) | `claude -p "what does main.py do?"` |
| Non-interactive + continue | `claude -c -p "fix the test"` |
| Limit agent turns | `claude --max-turns 3 -p "query"` |
| JSON output | `claude -p --output-format json "query"` |
| Add extra directories | `claude --add-dir ../lib ../shared` |
| Select model | `claude --model sonnet` |
| Show version | `claude --version` |
| Update CLI | `claude update` |
| Check health | `claude doctor` |

## Claude Code — Piping

```bash
cat error.log | claude -p "find the root cause"
git diff | claude -p "review these changes"
git log --oneline -20 | claude -p "summarize recent commits"
ls -la | claude -p "explain this directory"
```

## Claude Code — Configuration

```bash
claude config list                # view all settings
claude config get <key>           # check specific setting
claude config set <key> <value>   # change setting
claude config add <key> <value>   # push to array
claude config remove <key> <value>  # remove from array
```

**Settings priority** (highest → lowest):
1. Enterprise `/etc/claude-code/managed-settings.json`
2. Project local `.claude/settings.local.json` (git-ignored)
3. Project shared `.claude/settings.json` (team)
4. User global `~/.claude/settings.json`

## Claude Code — MCP Management

```bash
claude mcp add <name> <command> [args]         # add stdio server
claude mcp add --transport sse <name> <url>    # add SSE server
claude mcp list                                # show installed
claude mcp remove <name>                       # remove server
```

## Toolbox — Quick Install (curl one-liner)

```bash
curl -fsSL https://github.com/Adservio-Dev/AdservioToolbox/releases/download/v0.2.0/install.sh \
  -o install.sh && bash install.sh
```

Or with checksum verification:
```bash
curl -fsSL .../install.sh -o install.sh \
  && curl -fsSL .../SHA256SUMS -o SHA256SUMS \
  && sha256sum -c SHA256SUMS && bash install.sh
```

## Quick Reference — memctl

| Command | Purpose |
|---------|---------|
| `memctl init` | Initialize memory workspace |
| `memctl push "query" --source file` | Ingest files + recall context |
| `memctl pull --title "note"` | Store LLM output from stdin |
| `memctl search "query"` | Search memories (FTS5) |
| `memctl show <id>` | Show item details |
| `memctl stats` | Show store statistics |
| `memctl export` | Export as JSONL |

## Quick Reference — CloakMCP

| Command | Purpose |
|---------|---------|
| `cloak pack --policy P --dir D` | Sanitize secrets in directory (vaulted) |
| `cloak unpack --dir D` | Restore original content from vault |
| `cloak pack-file --policy P --file F` | Sanitize a single file |
| `cloak unpack-file --file F` | Restore a single file |
| `cloak scan --policy P --input F` | Scan for secrets (audit only) |
| `cloak guard --policy P` | Stdin guard (exit 1 if secrets) |
| `cloak vault-stats --dir D` | Show vault statistics |
| `cloak recover --dir D` | Recover from abnormal session exit |
